4 сценария ранжирования зон

⬜
Сценарий 1
Белые пятна
Высокий спрос + нет ни одного банкомата ВТБ. Лучший ROI с нуля: клиенты есть, инфраструктуры нет.
Ключ: vtb_atm_count = 0 AND demand_score ≥ P75
🏦
Сценарий 2
Перехват конкурента
Конкуренты присутствуют, ВТБ — нет. Перехват трафика Сбера / Альфа без органического спроса.
Ключ: competitor_atm_count ≥ 2 AND vtb_atm_count = 0
🏗️
Сценарий 3
Новые / старые ЖК·ТЦ
Динамика активности: рост = новое строительство, снижение = реновация / снос. Опережающий сигнал.
Ключ: growth_rate > +20% OR growth_rate < -20%
📉
Сценарий 4
Низкая загрузка
Уже установленные банкоматы ВТБ с падающим трафиком. Кандидаты на переброс или реконфигурацию.
Ключ: vtb_atm_count ≥ 1 AND txn_trend < -15%
⬜ Сценарий 1 — Белые пятна

Критерии и расчёт Demand Score

Зоны H3 (res.9) с высоким спросом и отсутствием любого банкомата ВТБ

📋 Фильтр-условия (обязательные)
1
vtb_atm_count = 0 — нет ни одного банкомата ВТБ
2
atm_activity = 0 — зона исторически не обслуживалась банкоматом ВТБ
3
unique_customers ≥ P50 — не менее медианного числа уникальных клиентов
⚖️ Веса компонентов Demand Score
Транзакции (total_sum)
35%
Уник. клиенты
25%
Транзакций на клиента
20%
Инфраструктура (POI)
12%
Конкурентный буфер
8%
DEMAND SCORE FORMULA
DS = 0.35 × norm(total_sum)
+ 0.25 × norm(unique_customers)
+ 0.20 × norm(txn_per_customer)
+ 0.12 × norm(poi_score)
+ 0.08 × norm(comp_proximity_bonus)
→ norm(x) = (x − min) / (max − min)
→ Threshold: DS ≥ 0.70 (топ-25%)
🗺️ Слои карты для этого сценария
Зелёный — DS ≥ 0.70, vtb=0 → Рекомендуется
Жёлтый — DS 0.5–0.70 → Наблюдение
Серый — DS < 0.50 → Низкий приоритет
Синий — уже установленный ВТБ

🏦 Сценарий 2 — Перехват конкурента

Критерии, расчёт и карта угроз

📋 Условия отбора
1
vtb_atm_count = 0 — ВТБ отсутствует в зоне
2
competitor_atm_count ≥ 2 — минимум 2 банкомата Сбер / Альфа / Тинькофф
3
vtb_clients_nearby ≥ 300 — клиенты ВТБ, вынужденные ходить к конкурентам
4
avg_competitor_distance ≤ 400м — буфер конкурентного присутствия
INTERCEPT SCORE
IS = 0.40 × norm(competitor_atm_count)
+ 0.30 × norm(vtb_clients_nearby)
+ 0.20 × (1 − norm(avg_comp_dist))
+ 0.10 × norm(mcc_cash_demand)
→ Шорт-лист: IS ≥ 0.75

🗺️ Слой карты: Конкурентная угроза
Тепловая карта: красный = высокий IS (Сбер/Альфа доминирует, нет ВТБ) · жёлтый = умеренная угроза · зелёный = ВТБ уже присутствует.
Иконки конкурентов кластеризуются по типу банка. Клик — всплывающая карточка с IS, расстоянием, числом клиентов ВТБ.
🏗️ Сценарий 3 — Новые / старые ЖК · ТЦ · БЦ

Динамический анализ активности зон

📈 РОСТ — Новое строительство
1
txn_growth_rate ≥ +20% (период T-3 → T, квартал)
2
new_accounts_growth ≥ +15% — приток новых клиентов ВТБ
3
construction_flag = 1 — ЖК/ТЦ/БЦ в радиусе 500м (внешний источник: Росреестр, 2ГИС)
Действие: Превентивная установка — до сдачи объекта ≥ 2 квартала
📉 СНИЖЕНИЕ — Реновация / снос
1
txn_decline_rate ≤ -20% (3 последовательных квартала)
2
renovation_flag = 1 — программа реновации Москвы (публичный слой ДГИ)
3
active_clients_delta ≤ -30% — активные клиенты за 12 мес.
Действие: Запланировать вывод банкомата до отключения зоны
GROWTH SIGNAL SCORE (GSS)
GSS = sign(growth_rate) × (
0.45 × |norm(txn_growth_rate)|
+ 0.30 × |norm(new_acc_growth)|
+ 0.25 × construction_weight
)
→ GSS ≥ +0.60: Зона роста 🟢
→ GSS ≤ −0.60: Зона упадка 🔴
🗺️ Слои карты
🟢 Зелёный градиент — GSS ≥ 0.60, рост: новые ЖК/ТЦ
🔴 Красный градиент — GSS ≤ −0.60, упадок: реновация
🏗️ Иконки объектов — строящиеся ЖК/ТЦ/БЦ из 2ГИС + Росреестра
📅 Ось времени — слайдер для просмотра изменений по кварталам
⚠️ Алерт-слой — зоны с bank_atm=1 и GSS ≤ −0.60 (приоритет вывода)
📉 Сценарий 4 — Низкая загрузка банкоматов ВТБ

Критерии выявления и действия

📋 Условия выявления
1
vtb_atm_count ≥ 1 — банкомат ВТБ установлен
2
txn_per_day ≤ P25 — транзакций/день ниже 25-го перцентиля
3
txn_trend_12m ≤ -15% — снижение за последние 12 месяцев
4
utilization_rate ≤ 30% — загрузка < 30% от нормативной
UNDERLOAD SCORE (US)
US = 0.40 × (1 − norm(txn_per_day))
+ 0.35 × |norm(txn_decline_12m)|
+ 0.25 × (1 − norm(util_rate))
→ US ≥ 0.70: Высокий риск простоя
→ Рекомендация: переброс / ребалансировка
⚡ Действия по результатам
A
Переброс в «Белое пятно» — связать US с DS: если рядом есть зона DS ≥ 0.80, предложить переброс
B
Ребалансировка — изменить режим работы (24/7 → дневной), снизить затраты
C
Мониторинг — 3 мес. наблюдение, повторная оценка после ремонта / реновации
0
банкоматов ВТБ с US≥0.70
0
рекомендованы к перебросу
